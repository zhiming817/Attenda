# Attenda

**出席即证明**

基于 Sui 智能合约、Walrus 存储和 Seal 加密的去中心化活动与票务平台。

## 概览
如今的活动报名平台（如 Luma、Eventbrite、Google Forms）依赖集中式数据模型和基于电子邮件的访问。门票、确认与活动详情都由中介方管理。

本项目旨在构建一个由 Sui 智能合约、Walrus 存储 与 Seal 加密 驱动的去中心化活动与票务平台，用户可以：

- 使用链上身份或 ZkLogin 注册。
- 接收可验证的、加密的 NFT 门票。
- 解锁托管在 Walrus Sites 上的活动详情访问权限。
- 通过出席获得出席 NFT 作为到场证明与领奖资格。

尽管本 RFP（征求建议书）列出了推荐的功能与集成方式，申请者仍然被鼓励提出可替代的架构、流程或更优的解决方案。欢迎有创造性、能改进或挑战本规范的方案，前提是与 Sui 与 Walrus 的能力相兼容。

支持目的

对于组织者：透明且可验证的活动数据与参与历史。
对于用户：真正拥有门票、私密加密访问以及通过出席建立声誉的能力。
对于生态系统：活动参与成为可组合的数据，可用于空投、忠诚计划和参与度洞察。

## 理想功能
活动创建与报名

- 在 Walrus 上存储带有元数据的活动（标题、时间、图片、描述）。
- 支持 ZkLogin 以简化用户入驻。
- 对于限额活动可选择审批流程。
- 灵活的票务参数（价格、容量、可见性等）。
- 成功报名后，用户会收到可验证的确认信息，确认还可包含可选的 ICS 日历文件（“添加到日历”）。
- 支付集成与折扣码管理。

NFT 门票

- 在 Sui 上铸造门票 NFT，并链接到 Walrus 的 blob。
- 门票 blob 包含被 Seal 加密的元数据（活动地点、二维码、访问链接）。
- 只有合法持有者可解密并查看详情。
- 可选的动态更新（状态、RSVP、出席信息）。

活动访问与验证

- 持票人可访问完整的 Walrus Site，内容包含活动议程、演讲嘉宾、媒体与公告等特定内容。
- 支持二维码或钱包验证入场。
- 在会场，门票 NFT 可被扫描以验证并进行出席记录。
- 出席追踪与参与证明记录上链。

出席验证

- 扫描确认出席后，可触发铸造出席 NFT（可选为灵魂绑定）。
- 出席证明日后可用于空投、后续活动或信誉系统。

通信

- 可选集成的通知（提醒、更新、确认、活动聊天）。
- 支持通过电子邮件或钱包发送通知（提醒、更新、确认）。

会后互动

- 活动数据在 Walrus 上保持可验证。
- 组织者可以导出匿名化的出席数据用于分析。
- 参与者可以在生态系统中展示他们的出席记录。
- 票务销售报告可用于税务用途。
- 用户保留对存储在 Walrus 上数据的控制权，可删除或撤销对其加密 blob 的访问，但链上的证明保持不变。

可选 / 扩展功能

- 潜在客户采集与社交网络功能，例如可选择的资料共享、基于钱包的声誉链接或会后跟进。
- 支持多分会场或多流程的报名，用于大型活动中不同票种、议程或参与者类型。
- 现场活动的胸牌打印与签到支持，包括基于二维码或 NFC 的访问验证。
- 面向活动应用、赞助商激活或忠诚度计划的集成点，利用链上出席数据实现更多玩法。
